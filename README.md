# Tandem-Cup 2026 – GitHub Pages

## Veröffentlichung

1. Bei GitHub ein neues Repository mit dem Namen `tandemcup.github.io` anlegen.
2. Alle Dateien aus diesem Ordner in das Repository hochladen.
3. Repository öffnen: **Settings → Pages**.
4. Unter **Build and deployment** einstellen:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Speichern.

Die Seite ist danach unter `https://tandemcup.github.io/` erreichbar.

## Ergebnisse aktualisieren

Die Datei `data.json` direkt bei GitHub bearbeiten:
- `lastUpdated` anpassen
- Punkte, Gewicht in Gramm und Fischanzahl eintragen
- Änderungen mit **Commit changes** speichern

Die Webseite lädt die Datei automatisch neu. Bei geöffneter Seite erfolgt alle 60 Sekunden eine Aktualisierung.
