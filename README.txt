UPLOAD AUF GITHUB

1. ZIP-Datei auf dem Handy oder PC entpacken.
2. Im GitHub-Repository die alten Dateien index.html, style.css, script.js, data.json und den alten assets-Ordner löschen.
3. Danach ALLE Dateien und den assets-Ordner aus diesem Paket in die oberste Ebene des Repositorys hochladen.
4. Änderungen mit "Neue Website" bestätigen.
5. GitHub Pages benötigt meist 1 bis 5 Minuten.

Wichtig: Nicht den äußeren Ordner hochladen. index.html muss direkt in der obersten Ebene des Repositorys liegen.
